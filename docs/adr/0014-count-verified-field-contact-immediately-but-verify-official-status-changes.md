# Count verified field contact immediately but verify official status changes

Requiring service-unit review for every valid household contact would create unnecessary workload, while changes to official vaccine status or registry eligibility alter district totals and require certification. A follow-up recorded by an individual field-worker account from an accepted confirmation source will count immediately for the contact/visit KPI; reports of vaccination, moving out, death, or other official-status changes will remain pending until verified by รพ.สต./PCU staff.
