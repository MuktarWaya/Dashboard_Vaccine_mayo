# Record alternative-vaccine service decisions without automated eligibility

Eligibility for the Pattani PAO-supported Hexaxim service depends on service-level clinical and consent considerations that responsible รพ.สต./PCU staff already assess and that should not be encoded as automatic dashboard eligibility rules. The system will let service-unit staff mark children as alternative-vaccine targets and confirm receipt of the supported vaccine so reporting can show which child groups benefited from PAO support, while keeping official vaccine-status certification separate.
