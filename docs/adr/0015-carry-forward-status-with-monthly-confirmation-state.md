# Carry forward official status with a separate monthly confirmation state

Re-entering unchanged vaccine statuses for thousands of children every month would add unnecessary workload, while treating copied values as confirmed would falsely imply monthly review. At the start of a reporting month, the system will carry forward each child's prior official vaccine status as an initial value and mark it `ยังไม่ยืนยันประจำเดือน` until the current month's review or confirmation is completed; carried-forward data alone cannot make a unit's submission complete.
